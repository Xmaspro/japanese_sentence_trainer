# Handover: Japanese Sentence Trainer

Date: 2026-05-30

## Current Goal

把原来的日语句子训练器改成以 5 个日常会话主题为主的训练工具。新闻功能暂时不继续开发，先保证生活/学习日语对话训练稳定可用。

## Main App

- Entry page: `japanese_sentence_trainer/index.html`
- Main logic: `japanese_sentence_trainer/app.js`
- Shared/testable core: `japanese_sentence_trainer/trainer_core.js`
- Course metadata/fallback corpus: `japanese_sentence_trainer/curriculum.js`
- Dialogue source corpus: `japanese-daily-dialogue-main/data/topic1.json` to `topic5.json`
- Converted training corpus: `content_archive/dialogue_items.json`

## How To Run On Another Computer

1. Clone and enter the repo.

   ```powershell
   git clone https://github.com/Xmaspro/japanese_sentence_trainer.git
   cd japanese_sentence_trainer
   ```

2. Start the local server.

   ```powershell
   node tools/serve_trainer_with_tts.js
   ```

3. Open:

   ```text
   http://127.0.0.1:5177/japanese_sentence_trainer/index.html
   ```

No npm install is currently required for the static app and tests.

## Important Recent Fix

The previous prompt `需要包含：会话表达` was a bug.

Root cause: imported dialogue items without a pattern were given the placeholder `会话表达`, and the replacement-mode checker treated it as a required phrase.

Current behavior:

- If a sentence has a real `patternNeedle`, 替换 mode checks that pattern.
- If it is ordinary dialogue without a pattern, 替换 mode accepts a sufficiently long Japanese sentence and does not require `会话表达`.
